# anotherTodoList

This is the README file for the "anotherTodoList" application.

Created by @mlinconnuKaliop and @lpoiret.

## Description

This application is a todo list built using React and Vite. It provides a minimal setup to get React working in Vite with HMR (Hot Module Replacement) and some ESLint rules.

## Official Plugins

- [@vitejs/plugin-react](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react/README.md): This plugin uses Babel for Fast Refresh.

## Usage

- Clone the repository.
- Install the dependencies using `npm install`.
- Start the development server using `npm run dev`.
- Open the application in your browser at `http://localhost:3000`.

Happy coding!
